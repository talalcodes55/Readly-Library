# Readly-Library
Electronic library that contains a wide variety of books to read online by a chosen plan , free or paid, the books could be downloaded with the paid plan.
This is only an interface and is not completed library and has not been linked to a back-end operations and databases yet, this is just a demonstration of the library made with HTML, CSS, Bootstrap, and this library will be improved in the future.
